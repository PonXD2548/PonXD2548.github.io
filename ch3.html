7. มาถึงหน้าที่สำคัญ นั่นก็คือ การแบ่ง Partition หรือการแบ่งไดรฟ์ นั่นเอง พูดง่าย ๆ ก็คือ แบ่งพื้นที่ไดรฟ์ C และไดรฟ์ D ที่เราใช้งานนั่นแหละ โดยจะมีอยู่ 2 กรณีดังนี้

1. ยังไม่ได้ติดตั้ง Windows : หากเป็นคอมพิวเตอร์เครื่องใหม่ ในหน้านี้จะมี Drive 0 Unallocated Space เท่านั้น ให้ คลิก Drive 0 Unallocated Space > New > แบ่งไดรฟ์ตามต้องการ (โดย 1024Mb = 1Gb ดังนั้นถ้าหากต้องการแบ่ง ไดรฟ์ C ให้มีพื้นที่ 200GB คุณก็ต้องนำ 1024 × 200 = 204800MB) > คลิก Apply ซึ่งถ้าหากต้องการแบ่งไดรฟ์อื่น ๆ อีกก็ทำเหมือนเดิม เมื่อแบ่งเสร็จแล้ว คลิก Next

2. ติดตั้ง Windows มาก่อน : แต่หากเป็นคอมพิวเตอร์ที่มีการติดตั้ง Windows มาก่อน ในหน้าก็จะมีการแบ่งไดรฟ์เอาไว้แล้ว ให้ทำการ คลิก ไดรฟ์ C (ในตาราง Type จะเป็น System) > คลิก Format เพื่อที่จะลบข้อมูลวินโดว์เก่าทิ้ง แล้ว คลิก Next
